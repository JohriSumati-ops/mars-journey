# 🚀 Mars Journey — Interactive Web Experience

An immersive, cinematic journey from Earth to Mars, built for the Frontend Odyssey Hackathon

---

## 🌌 Overview
Mars Journey is a story-driven interactive web experience that simulates a futuristic space mission. Instead of a traditional website, it presents a narrative journey where users travel from Earth to Mars through smooth scrolling, animations, and immersive audio.

The project focuses on combining UI/UX, storytelling, and interactivity to create a visually engaging and emotionally rich experience.

---

## ✨ Features
#### 🎬 Cinematic Entry Experience
Begin your journey with an engaging “Begin Mission” screen.
#### 🌍➡️🪐 Scroll-Based Space Travel
Seamless transition from Earth to Mars using storytelling through scroll.
#### 🎧 Immersive Audio Experience
Background music enhances the cinematic feel of the journey.
#### 🚀 Interactive Mission Selection
Users can select missions and view details like duration and objectives.
#### 📱 Fully Responsive Design
Optimized for mobile, tablet, and desktop devices.
#### 🎨 Modern UI/UX Design
Clean layout with consistent space-themed visuals.
